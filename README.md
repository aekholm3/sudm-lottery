# DO NOT PUSH ATTENDEES OR TRANSACTIONS FILE TO GITHUB

**TO USE THIS CODE:**

- Clone repository onto local

- Download attendees file and transactions file and place in same directory as handler.py (this will be the directory of the cloned respository)

- Rename ATTENDEES_FILE in handler.py to be the name of the new attendees file

- From command line `python3 handler.py`
